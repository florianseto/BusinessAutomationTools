# Project001 CSV Excel Automation Tool

This project is a business automation tool for processing CSV data and exporting structured Excel reports.

## Purpose

To reduce manual work by automating CSV data processing, Excel report generation, and basic business data organization.

## Planned Features

- Import CSV files
- Clean and process data
- Export Excel reports
- Generate summary sheets
- Save processing logs

## Status

Planning
